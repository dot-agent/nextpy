### ü§£ Creazione di un'App Generatore di Barzellette (5 minuti)

Modifica `myapp/myapp.py` per creare la tua app.

#### 1. Importa le Librerie

```python
import nextpy as xt
import pyjokes
```

Inizia importando `nextpy` per lo sviluppo e `pyjokes` per barzellette casuali.

#### 2. Configura lo Stato dell'App

```python
class State(xt.State):
    joke: str = "Clicca il bottone per ottenere una barzelletta!"

    def generate_joke(self):
        self.joke = pyjokes.get_joke()
```

Crea una classe `State` con una variabile `joke`. Usa `generate_joke` per recuperare nuove barzellette.

#### 3. Progetta la Pagina Principale

```python
def index():
    layout = xt.vstack(
        xt.text(State.joke, font_size="2em"),
        xt.button("Genera Barzelletta", on_click=State.generate_joke),
    )
    return layout
```

La funzione `index` organizza la visualizzazione della barzelletta e un bottone. `vstack` √® usato per impilare i componenti verticalmente, mentre `hstack` √® usato per impilarli orizzontalmente.

#### 4. Stilizzazione Opzionale
```md
def index():
    layout = xt.vstack(
        xt.text(State.joke, font_size="2em"),
        xt.button("Genera Barzelletta", on_click=State.generate_joke),

        spacing="1em",
        align_items="center",
        justify_content="center",
        height="100vh",
   )
    return layout
```
L'attributo `spacing` aggiunge spazio tra gli elementi `text` e `button`, mentre `align_items` e `justify_content` assicurano che questi elementi siano centrati. L'altezza dello stack `height` √® impostata al 100% dell'Altezza del Viewport (`100vh`), il che permette allo stack verticale di riempire interamente l'altezza dello schermo.

#### 5. Configura l'app

```python

app.add_page(index)

```

Configura l'app, aggiungi la pagina principale. Per visualizzare il frontend, naviga su `localhost:3000`.
